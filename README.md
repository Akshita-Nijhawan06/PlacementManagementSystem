# Placement Management System

## Overview
A Java-based Placement Management System developed using JDBC and MySQL.

## Features
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student

## Technologies Used
- Java
- JDBC
- MySQL
- IntelliJ IDEA

## Database
Database Name: placement_db

Table: students

## Project Structure

src/
├── app/
│   └── PlacementManagementSystem.java
├── dao/
│   └── StudentDAO.java
├── db/
│   └── DatabaseConnection.java
└── model/
    └── Student.java
